- **RLS Latency**: Complex RLS policies can impact query performance. Always test with `EXPLAIN ANALYZE`.
- **Foreign Key Locks**: High-frequency updates to rows with many foreign key constraints can lead to lock contention.
- **Migration Drift**: Never skip a migration version. Sequential integrity is mandatory.
