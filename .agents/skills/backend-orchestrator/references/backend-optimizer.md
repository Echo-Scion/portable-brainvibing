1. **Bottleneck Source**: Determine if issue is I/O Bound or CPU Bound.

## 🔍 Troubleshooting (Quick Guide)
- **OOM Crash**: Massive arrays in memory. Refactor to cursors/streams.
- **Latency Spikes**: Event Loop Blockage. Move heavy tasks to worker threads.
- **502/504 Errors**: External API hang. Review timeout limits.
